Quien soy?

¡Hola! Soy Alexis, me dedico al diseño y desarrollo web. Un entusiasta de Java y un tipo ligeramente obsesionado por la calidad del código. Actualmente estoy disponible para trabajar como autónomo. Si tienes un proyecto que quieres comenzar o crees que necesitas mi ayuda con algo, envíame un mensaje.

Que puedo hacer?

Front-end

Como desarrollador de JavaScript, Tengo experiencia en técnicas HTML5 y CSS3. trabajando con jQuery o javascript MVC más avanzado marcos como angulares

Back-end

Utilizando marcos php como Zend o creando código personalizado, he escrito servicios que brindan soporte a miles de usuarios, incluidas API REST, aplicaciones de aprendizaje electrónico y más.

Consultancy

Además de brindar servicios de desarrollo, también puedo ayudarlo a decidir hojas de ruta estratégicas a través de servicios de consultoría.services.
